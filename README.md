## Theory  

### 1. Text Data  
Text data is unstructured information in the form of words, sentences, or documents (emails, social media posts, reviews, articles, etc.).  
Before applying machine learning, text must be cleaned and processed — this is called **text preprocessing**.  

### 2. Preprocessing Techniques  
- **Text Cleaning:** Remove punctuation, special characters, extra spaces, irrelevant symbols.  
- **Lowercasing:** Convert all text to lowercase for consistency.  
- **Tokenization:** Split text into tokens (words/sentences).  
- **Stop Word Removal:** Remove common words (is, the, and, in, of).  
- **Stemming:** Reduce words to root form (e.g., *playing → play*).  
- **Lemmatization:** Convert words to dictionary form (e.g., *studies → study*).  

### 3. NLP Techniques  
- **Tokenization:** Divide text into words/sentences.  
- **POS Tagging:** Identify grammatical categories (Noun, Verb, Adjective).  
- **Named Entity Recognition (NER):** Detect entities like names, locations, organizations, dates.  
- **Sentiment Analysis:** Determine emotion/opinion (Positive, Negative, Neutral).  
- **Text Vectorization:** Convert text into numerical format (BoW, TF-IDF).
